# Group-2-Project

## Getting Started

These instructions will get your project up and running on a local machine.

### Dependencies

List any prerequisites, libraries, operating system versions, etc., needed before installation.

- Node.js (version 14+)
- npm (version 6+)
- Git

### Installation

Step-by-step instructions on how to install and set up your application.

1. Clone the repository:

   ```bash
   git clone https://github.com/Group-2-2/Group-2-Project.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Group-2-Project
   ```

3. Install NPM packages:

   ```bash
   cd backend
   npm install
   cd ../frontend
   npm install
   ```

### Executing Program

Instructions on how to run the project once installed.

#### Frontend

1. Start the frontend server:

   ```bash
   cd frontend
   npm run dev
   ```

2. Visit `http://localhost:3000` in your browser to use the application.

#### Backend

1. Start the backend server:

   ```bash
   cd backend
   nodemon server.js
   ```
